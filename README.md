GhostHub is a Python tool designed for ethical hacking education and testing purposes. It allows you to change DNS addresses and connect to the Tor network, providing enhanced privacy. The tool also includes the ability to launch the Tor Browser.

## Usage

Follow these steps to use the project:

1. **Download Requirements:**

   - Run the following command in the terminal to install the required dependencies listed in `requirements.txt`:

   ```bash

   Run the Project:
   ```
